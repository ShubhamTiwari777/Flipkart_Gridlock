GRIDLOCK — Event-Driven Traffic Incident Intelligence
Flipkart Hackathon Round 2 | Problem Statement 2
======================================================

FIRST TIME SETUP
-----------------

S1 — Extract zip
    Right-click the zip → Extract All → choose any folder you like.
    You will get a folder called: gridlock_event_driven_congestion

S2 — Open CMD inside that folder
    Open the extracted gridlock_event_driven_congestion folder in File Explorer.
    Click the address bar at the top → type "cmd" → press Enter.
    A Command Prompt will open already inside that folder.

S3 — Install deps
    pnpm install
    (takes 1–2 minutes, needs Node 18+ and pnpm installed)
    Get Node: https://nodejs.org
    Get pnpm: npm install -g pnpm

S4 — Approve esbuild (only if you see this warning)
    Warning: [ERR_PNPM_IGNORED_BUILDS] Ignored build scripts: esbuild@0.27.3

    Run:
        pnpm approve-builds
    Arrow keys → select esbuild → Space → Enter → then run pnpm install again
    Skip this step if you did not see the warning.

S5 — Fix Rollup (Windows only, only if you see this error)
    Error: Cannot find module @rollup/rollup-win32-x64-msvc

    Run:
        pnpm add -D @rollup/rollup-win32-x64-msvc --filter @workspace/gridlock-dashboard
    Skip this step if you did not see the error.

NOTE: The .env file is already included — no map key setup needed.


RUNNING THE APP  (need 2 terminal windows open at the same time)
-----------------------------------------------------------------

TERMINAL 1 — Start API server
    (use the CMD you opened in S2)

    PowerShell (prompt starts with PS):
        $env:NODE_ENV = "development"
        $env:PORT = "5000"
        pnpm --filter @workspace/api-server run build
        pnpm --filter @workspace/api-server run start

    CMD (prompt looks like C:\>):
        set NODE_ENV=development
        set PORT=5000
        pnpm --filter @workspace/api-server run build
        pnpm --filter @workspace/api-server run start

    Wait for: INFO  Server listening  port: 5000
    Keep this window open.

TERMINAL 2 — Start dashboard
    Open a second CMD the same way (File Explorer → address bar → "cmd" → Enter)

    pnpm --filter @workspace/gridlock-dashboard run dev

    Wait for:
        VITE v7.x.x  ready in xxxx ms
        Local: http://localhost:3000/

S6 — Open dashboard in browser
    http://localhost:3000

    Pages:
        /            Overview (KPI cards + hotspot map + events table)
        /predict     Event Predictor (real-time 3-model ML)
        /analytics   Corridor + temporal + cause charts
        /pipeline    ML architecture + model metrics
